# goKuka
Generates Kuka Robot Language from spline points described in CSV

## usage
```
 goKuka [global options] command [command options] [arguments...]

COMMANDS:
     random, r  random -n 40
     repeat, p  repeat -n 30
     csv, c     csv `FILE` -m 50

```
## sample
```
goKuka csv 'sampleSpline.csv'
```
